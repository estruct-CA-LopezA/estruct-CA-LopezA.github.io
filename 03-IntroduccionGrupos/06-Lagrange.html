Teorema de Lagrange
Si \(G\) es un grupo finito y \(H \le G\), entonces \( \left . \operatorname{Ord} \left ( H \right ) \, \middle \vert\, \operatorname{Ord} \left ( G \right ) \right . \) y \[ \left [ G:H \right ] =\frac{ \operatorname{Ord} \left ( G \right ) }{ \operatorname{Ord} \left ( h \right ) }\]

Demostración

A

Sea \(\mathcal{R}\) la familia de todas las clases derechas de elementos de \(H\) en \(G\). Observe que \(\mathcal{R}\subseteq \mathcal{P} \left ( G \right ) \), de manera que tanto \(\mathcal{R}\) como sus elementos son finitos. Más aún, \(\mathcal{R}\) es una partición de \(G\) por lo visto en
Lema 13
y la
Observacion 16
. ​
Como \(G\) y \(R\) son finitos, de cada una de las clases las clases de equivalencia de \( \overset{de}{\equiv}_{H} \) en \(G\), es decir, de cada clase lateral derecha, podemos escoger exactamente un representante. Sean \[a_{1},\dots,a_{n}\] dichos representantes, los cuales forman un sistema completo de representantes al ser las clases laterales disjuntas entre sí, de manera que: \[\begin{align*} \mathcal{R} &= \left.\left.\Rule{0em}{1.5em}{0em}\right \{ H a_{1} ,\dots, H a_{n} \right.\left.\Rule{0em}{1.5em}{0em}\right \} \\ \Cardinalidad{\mathcal{R}} &=n\\ &= \left [ G:H \right ] \end{align*}\] Finalmente, observe que para cada \(i\in \left.\left.\Rule{0em}{1.5em}{0em}\right \{ 1,\dots,n \right.\left.\Rule{0em}{1.5em}{0em}\right \} \), se cumple que \(\Cardinalidad{H}=\Cardinalidad{ H a_{i} }\) por el
Lema 20
. \[\begin{align*} \operatorname{Ord} \left ( G \right ) &=\Cardinalidad{G}\\ &=\Cardinalidad{\bigcup\mathcal{R}}\\ &=\Cardinalidad{ \left.\left.\Rule{0em}{1.5em}{0em}\right \{ H a_{1} ,\dots, H a_{n} \right.\left.\Rule{0em}{1.5em}{0em}\right \} }\\ &=\Cardinalidad{\bigcup_{i=1}^{n} H a_{1} }\\ &=\sum_{i=1}^{n}\Cardinalidad{ H a_{1} }\\ &=\sum_{i=1}^{n}\Cardinalidad{H}\\ &=n\Cardinalidad{H}\\ &= \left [ G:H \right ] \operatorname{Ord} \left ( H \right ) \end{align*}\] Lo anterior nos indica que \( \left . \operatorname{Ord} \left ( H \right ) \, \middle \vert\, \operatorname{Ord} \left ( G \right ) \right . \) y \[ \left [ G:H \right ] =\frac{ \operatorname{Ord} \left ( G \right ) }{ \operatorname{Ord} \left ( h \right ) }\]
square

Corolario

2

Sea \(G\) un grupo finito. Entonces cualquier elemento de \(G\) es de orden finito y \( \left . \operatorname{Ord} \left ( a \right ) \, \middle \vert\, \operatorname{Ord} \left ( G \right ) \right . \)

Demostración

A

Como \( \left \langle a \right \rangle \le G\), \( \left \langle a \right \rangle \subseteq G\), por lo cual \(\Cardinalidad{ \left \langle a \right \rangle }\subseteq \Cardinalidad{G}<\infty\). Por el
Teorema 1 Teorema de Lagrange
, se cumple que \( \left . \operatorname{Ord} \left ( \left \langle a \right \rangle \right ) \, \middle \vert\, \operatorname{Ord} \left ( a \right ) \right . \). Además, como \( \operatorname{Ord} \left ( \left \langle a \right \rangle \right ) = \operatorname{Ord} \left ( a \right ) \), se cumple que \[ \operatorname{Ord} \left ( \left \langle a \right \rangle \right ) \le \operatorname{Ord} \left ( G \right ) <\infty \land \left . \operatorname{Ord} \left ( a \right ) \, \middle \vert\, \operatorname{Ord} \left ( G \right ) \right . \]
square

Corolario

3

Sea \(G\) un grupo finito y \(a\in G\), entonces \(a^{ \operatorname{Ord} \left ( G \right ) }=e\), donde \(e\) es el elemento neutro de \(G\).

Demostración

A

Por
Corolario 2
\( \left . \operatorname{Ord} \left ( a \right ) \, \middle \vert\, \operatorname{Ord} \left ( G \right ) \right . \), es decir \[\begin{align*} &\exists k\in \mathbb{Z} : \operatorname{Ord} \left ( G \right ) =k\cdot \operatorname{Ord} \left ( a \right ) \\ &\implies a^{ \operatorname{Ord} \left ( G \right ) }=a^{k\cdot \operatorname{Ord} \left ( a \right ) }=(a^{ \operatorname{Ord} \left ( a \right ) })^{k}=e^{k}=e\\ &\implies a^{ \operatorname{Ord} \left ( G \right ) }=e \end{align*}\] Así podemos concluir que \(a^{ \operatorname{Ord} \left ( G \right ) }=e\).
square
