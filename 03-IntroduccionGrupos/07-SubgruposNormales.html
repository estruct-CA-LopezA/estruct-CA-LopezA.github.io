Subgrupos normales

Definición

1

Subgrupo normal
Sea \((G,\cdot)\) un grupo, y sea \(H \le G\). se dice que \(H\) es un subgrupo normal de \(G\), lo cual se denota por \(H \trianglelefteq G\), sí y sólo sí \[\forall g\in G: gHg^{-1} \subseteq H\]

Lema

2

Sea \( \varphi : G_{1} \longrightarrow G_{2} \) un homomorfismo de grupos, entonces \( \operatorname{Ker} \left ( \varphi \right ) \) es es un subgrupo normal de \(G_{1}\).

Demostración

A

Veamos que \[\forall g\in \operatorname{Ker} \left ( \varphi \right ) : g \operatorname{Ker} \left ( \varphi \right ) g^{-1} \subseteq \operatorname{Ker} \left ( \varphi \right ) \] En efecto, sea \(g\in G_{1}\) arbitrario pero fijo, y sea \(a\in g \operatorname{Ker} \left ( \varphi \right ) g^{-1}\). Entonces, existe \(h\in \operatorname{Ker} \left ( \varphi \right ) \) tal que \(a=ghg^{-1}\). Observe que \[\begin{align*} \varphi(a) &=\varphi(ghg^{-1})\\ &=\varphi \left.\left.\Rule{0em}{1em}{0em}\right ( (gh)g^{-1} \right.\left.\Rule{0em}{1em}{0em}\right ) \\ &=\varphi(gh)\varphi \left.\left.\Rule{0em}{1em}{0em}\right ( g^{-1} \right.\left.\Rule{0em}{1em}{0em}\right ) \\ &=\varphi(g)\varphi(h)\varphi \left.\left.\Rule{0em}{1em}{0em}\right ( g^{-1} \right.\left.\Rule{0em}{1em}{0em}\right ) \\ &=\varphi(g)e_{2}\varphi \left.\left.\Rule{0em}{1em}{0em}\right ( g^{-1} \right.\left.\Rule{0em}{1em}{0em}\right ) \\ &=\varphi(g)\varphi \left.\left.\Rule{0em}{1em}{0em}\right ( g^{-1} \right.\left.\Rule{0em}{1em}{0em}\right ) \\ &=e_{2} \end{align*}\] ​i.e. \(a=ghg^{-1}\in \operatorname{Ker} \left ( \varphi \right ) \), por lo cual \(gHg^{-1}\subseteq H\), y esto se cumple para cada \(g\in G\). Así, \( \operatorname{Ker} \left ( \varphi \right ) \subseteq G\)
square

Proposición

3

Sea \(G\) un grupo conmutativo, entonces todo subgrupo \(H\) de \(G\) es normal.

Demostración

A

En efecto, sea \(H\) un subgrupo de \(G\). \(g\in G_{1}\) arbitrario pero fijo, y sea \(a\in gHg^{-1}\). Entonces, existe \(h\in H\) tal que \(a=ghg^{-1}\). Como \(G\) es conmutativo \[a=ghg^{-1}=gg^{-1}h=eh=h\in H\] así \[\forall g\in G \forall h \in H: ghg^{-1}\in H\] por lo cual \(gHg^{-1}\subseteq H\), y en consecuencia \(H \trianglelefteq G\), para cualquier subgrupo \(H\) de \(G\).
square
